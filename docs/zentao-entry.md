# 禅道需求记录模板

## 需求标题

成语检索工具 MVP

## 需求描述

实现一个轻量化成语检索工具，技术方案为 Rust + SQLite。成语数据文件随代码一起提交到 GitHub，用户 clone 仓库后无需部署外部数据库、Redis 或后端服务，即可在本地查询成语。

第一版支持：

- 精确查询成语。
- 按关键词模糊查询成语、拼音、释义、出处、典故、例句和标签。
- 查询结果展示读音、释义、出处、典故、例句、冷僻程度和标签。
- 包含 100 条以上成语数据，覆盖常见成语和部分较少见/冷僻成语。
- README 提供运行方式、查询示例、数据字段说明和数据来源说明。

## 验收标准

- GitHub 仓库可访问。
- `data/idioms.sqlite` 已随代码提交。
- clone 后执行 `cargo run -- exact 画蛇添足` 可以返回完整查询结果。
- 执行 `cargo run -- search 多此一举` 可以模糊查到相关成语。
- 执行 `cargo run -- list --rare` 可以列出较少见或冷僻成语。
- README 可以指导新用户直接运行项目。
